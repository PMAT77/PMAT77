# 围墙、影子和世界尽头

朝闻道

Nov 07, 2019 Paolos

早年读大刘的《朝闻道》，大致是说物理学家们试图制造创世级高能粒子加速器来得到宇宙的大统一模型，却被他们宇宙更高级的文明“排险者”阻止，这种级别加速器的存在会可能导致宇宙的衰变。

作为补偿，“排险者”许诺可以满足这些屹立科学尽头的可怜们一个获取宇宙任意真相的机会，但因为“知识密封准则”，排险者不能向人类传授超出地球现有文明水平的知识，要想获取真理，就要付出生命的代价。

为了洞悉宇宙的终极之美，哪怕只有短短一瞬，大批不同领域的顶尖学者们放弃了本该富足的生活、温暖的家庭，欣然成为真理的殉道者，他们一批批走向真理祭坛，在换取真理之后坦然化作火球归于灰烬。

任何对我们来说所谓的意义在宇宙的宏观角度上都没有意义，现实没有排险者出现，我想这也是那么多科学巨擘在晚年迷信神学的原因，遥遥无期的真相可能比任何存在都要黑暗今人绝望。

无生的喜悦，无死的恐惧。“朝闻道夕死可矣”。

其实就是用天文尺度的庞大格局满足一粒沙的毕生追求。
