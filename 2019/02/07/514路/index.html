# 围墙、影子和世界尽头

514路

Feb 07, 2019 Paolos

下午回富阳拿点东西，坐在514上，光线从侧边车窗懒懒的照进来，连月不见的太阳难得出了次全勤。

车厢里以外有些闷热。

菩提寺到富阳公交站正在修地铁，车多路窄，幽灵堵车。车流行的很慢，加上道路坑洼，一路上颠簸不止。

我趴在前座靠椅上，双臂枕着下巴，昏昏沉沉地犯困。

车行至受降，猛地来了个颠簸，屁股不受控制的腾飞，差点没磕掉牙。

司机嘀咕着抱怨，有乘客窃窃私语。

抬起头，车流仍是缓慢。
